# Código de conducta

Este proyecto se compromete a proporcionar un entorno libre de acoso para todos, independientemente de su género, orientación sexual, discapacidad, apariencia física, origen étnico, religión o cualquier otra condición.

No se tolerará ningún tipo de discriminación, insultos o comportamientos agresivos. Los moderadores del proyecto se reservan el derecho de eliminar contribuciones que no respeten este código.

Para reportar un problema, contacta a josemonteromunoz@gmail.com.
